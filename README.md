# 🚀 Ganesh Portfolio Website

A modern, futuristic, dark-themed developer portfolio built using Next.js, TypeScript, Tailwind CSS, and Framer Motion.

Inspired by premium UI/UX portfolio designs featuring neon glow effects, smooth animations, and recruiter-friendly presentation.

---

## 🎯 Project Goal

Create a professional portfolio website that showcases:

* About Me
* Skills
* Projects
* Education
* Certifications
* Resume
* Contact Information

Designed specifically for:

* Software Developer Roles
* ASP.NET Core Developer Roles
* BFSI Technology Roles
* Backend Developer Positions
